    \section{Cinco unidades idênticas são arranjadas em uma redundância ativa para formar um subsistema. As falhas das unidades são independentes, e pelo menos duas das unidades devem sobreviver 1000 horas para que o subsistema realize sua missão.}
    \paragraph{}
    \hspace{8pt}a) Se as unidades tem distribuição exponencial do tempo de falha, com taxa de falhas 0,002, qual é a confiabilidade do subsistema?
    
    \begin{center}
    $\lambda=0.002$\\
    $F(t)=1-e^{-\lambda t}$\\
    $R(t)=1-F(t)=e^{-\lambda t}$\\
    \end{center}    
    
    \hspace{8pt}Probabilidade de uma uinidade durar mais de 1000 horas é $R(1000)=0.1353$
    
    \hspace{8pt}Considerando $P(k)$ a probabilidade de sucesso. No sistema com 5 unidades a probabilidade de pelo menos duas não falharem é $P(k>2)=1-P(k=1)+P(k=0)$.
    
    \hspace{8pt}Como o sistema esta em redudancia é considerado que ele está em paralelo assim a probabilidade de todos falharem é $(1-R(1000))^5$ de apenas 1 falhar é $R(1000)(1-R(1000))^4*5$ no caso geral pela distribuição binimial $\binom{n}{k}(p)^k(1-p)^{n-k}$.
    
    \hspace{8pt}Portanto a confiabilidade do sistema é:
    \begin{center}
    $R(t)=P(k>2)=1-(P(k=1)+P(k=0))=1-((1-R(1000))^5+R(1000)(1-R(1000))^4*5)=0.138431685054$    
    \end{center}
    
    \hspace{8pt}b) Qual é a confiabilidade se for exigida a sobrevivência de apenas uma das unidades?
    
    \hspace{8pt}Considerando agora que pra o sistema funcionar basta uma unidade durar mais de 1000 horas.
    \begin{center}
    $R(t)=P(k>1)=1-P(k=0)=1-(1-R(1000))^5=0.516675635853$
    \end{center}
